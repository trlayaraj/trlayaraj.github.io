Projetcs

Student Management System
Description:

An Angular + ASP.NET Web API Student Management System, with JWT-based role login (Admin/Teacher/Student), role-based routing, admin dashboard backed by a BehaviorSubject for live counts, solved duplicate-subscription leaks by adding an initialization flag in the service, and clean backend-frontend contracts using DTOs.


Key learnings and findings:
  • Used lazy loading in angular frontend for performance
  • Implemented JWT authentication and Angular route guards for security
  • Used behavioursubject for live counts of departments, teachers, subjects etc.
  • Added extra flag to prevent unwanted subscription of readonly observable that leads to memory leak when subscribed in oninit() of other components.
  • While assigning subjects to teacher the dropdown list of subject contains only the relevant subject to that selected teahcer. (Not showing all the subjects available)
